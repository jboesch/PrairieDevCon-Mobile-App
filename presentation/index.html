Sencha Touch

Building an HTML5 Mobile App.

What is it?

Sencha Touch 2 is the only framework that enables developers to build fast and impressive apps that work on iOS, Android, BlackBerry, Kindle Fire, and more.

Things that make it awesome

  • Charting API
  • Event driven
  • MVC(S)
  • Easily themeable via SASS
  • Easy to compile for iOS or Android store
  • Free!

Things you need to develop

  • Modern webkit browser like Safari or Chrome
  • A webserver running locally

WebSockets to the rescue!

User connects with a single socket

  • Handshake happens between client and server.
  • Allows for full-duplex communication between client and server.

The code

var ws = new WebSocket('ws://localhost');

ws.onopen = function(){
    // do something when the socket connects successfully
}
ws.onmessage = function(evt){
    // message is contained in evt.data
}
ws.onerror = function(){
    // do something when an error occurs
}
ws.onclose = function(){
    // do something when the socket is closed
}

What browsers support native WebSockets?

  • Google Chrome 14+
  • Safari 5+
  • Firefox 6+
  • Opera 11+
  • iOS 4.2+
  • Internet Explorer 6... AHH just kidding. IE 10.

Fallbacks for older browsers?

Socket.IO (www.socket.io) for NodeJS

Server side

var io = require('socket.io').listen(80);

io.sockets.on('connection', function (socket) {
    socket.on('message', function () { });
    socket.on('disconnect', function () { });
});

Client side

var socket = io.connect('http://localhost/');
socket.on('connect', function () {
    socket.on('message', function (msg) {

    });
    socket.send('hi');
});

Current state

The WebSocket API is being standardized by the W3C, and the WebSocket protocol has been standardized by the IETF as RFC 6455.
